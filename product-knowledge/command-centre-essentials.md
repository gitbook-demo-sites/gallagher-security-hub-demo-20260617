# Command Centre essentials

Command Centre brings access control, alarms, perimeter events, and operator workflows into one operational view.

## Common tasks

- Review access events for a cardholder.
- Acknowledge and close an alarm.
- Update an access group.
- Check controller health before a site visit.
- Escalate a suspected security incident.
