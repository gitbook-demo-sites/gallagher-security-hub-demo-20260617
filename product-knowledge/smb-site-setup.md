# SMB site setup

This dummy page represents public setup guidance for smaller organizations rolling out Gallagher SMB Security.

## First week checklist

- Add the primary site administrator.
- Confirm door schedules.
- Test cardholder access at a low-risk door.
- Verify alarm notifications.
- Save local support contacts.

{% hint style="warning" %}
Do not reuse administrator accounts across staff members. Create named users so activity remains auditable.
{% endhint %}
